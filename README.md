# Schallwellen Labor

Eine kleine Browser-App, die Mikrofon-Audio live als Schallwelle und Frequenzspektrum sichtbar macht.

## Online nutzen

Diese App ist fuer GitHub Pages vorbereitet. Die Hauptdatei heisst `index.html`.

Nach dem Upload zu GitHub kann die Seite unter einer Adresse wie dieser laufen:

`https://DEIN-NAME.github.io/schallwellen-labor/`

Der Browser fragt beim Start nach Mikrofon-Zugriff.
